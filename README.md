## 🇹🇷 Türkçe

### Nedir?
PM Macro, sol ve sağ tık için bağımsız CPS (saniyedeki tıklama) ayarı yapabileceğin hafif bir Windows makro aracıdır. Kurulum gerektirmez, tek dosyadır.

### Özellikler
- 🖱️ Sol ve sağ tık için bağımsız CPS ayarı (1 – 1000)
- ⏸️ Master Durdur / Başlat butonu
- 🔁 Toggle modu + Basılı tut modu
- 📊 Gerçek zamanlı tıklama istatistikleri
- 🏆 CPS rekor takibi
- 🌐 Türkçe / İngilizce dil desteği
- 📦 Tek dosya, kurulum yok

### Kontroller
| Tuş | Fonksiyon |
|-----|-----------|
| `F6` | Sol tık toggle |
| `F7` | Sağ tık toggle |
| `F8` | Tümünü durdur / başlat |

### Kurulum
1. [Releases](../../releases) sayfasından `MACRO.exe` indir
2. Çalıştır — kurulum yok

## 🇬🇧 English

### What is it?
PM Macro is a lightweight Windows mouse macro tool that lets you set independent CPS (clicks per second) for left and right mouse buttons. No installation required, single file.

### Features
- 🖱️ Independent CPS (1 – 1000) for left & right click
- ⏸️ Master Stop / Start button
- 🔁 Toggle mode + Hold mode
- 📊 Real-time click statistics
- 🏆 CPS record tracker
- 🌐 Turkish / English language support
- 📦 Single file, no install

### Controls
| Key | Function |
|-----|----------|
| `F6` | Left click toggle |
| `F7` | Right click toggle |
| `F8` | Stop / Start all |

### Installation
1. Download `MACRO.exe` from [Releases](../../releases)
2. Run it — no installation needed
